③ 添加签名

建议使用透明背景的 PNG 图片

输入证书主题 (CN), 每次签名时会自动生成对应的 RSA-2048 自签名证书:

签名后将在 PDF 末尾追加独立签名页 显示证书详情, 元数据也会记录 (用 PDF 阅读器"文档属性→描述"查看)。

说明：本工具会将签名图章（文字/图片/内置任选其一）嵌入到 PDF 选定位置。

关于"数字签名"：符合 PDF 行业标准 (eIDAS / 21 CFR Part 11) 的数字签名需要 X.509 证书和 PKCS#7/CAdES 格式， 纯浏览器内实现复杂。本工具通过 Web Crypto API 提供的 "RSA + SHA-256 字节签名" 能保证 内容完整性 (检测 PDF 是否被篡改)，但不具备法律效力。
